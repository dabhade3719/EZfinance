# EZfinance – Personal Financial Assistant
📌 Project Overview

EZfinance is a web-based personal finance management application designed to help users improve financial literacy and manage their finances effectively. The application provides guidance on budgeting, savings, investments, and financial planning through a simple and user-friendly interface.


- 🚀 Features
- 💰 Budget and expense management
- 📈 Financial planning guidance
- 📊 Investment awareness information
- 📱 Responsive and user-friendly design
- 🔒 Secure and simple web application structure
- ⚡ Fast and lightweight interface


🛠️ Technologies Used
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Server: WAMP Server
- Tools: Visual Studio Code, Google Chrome, Microsoft Edge
